# 快速入门 {#concept_odx_wwf_cfb .concept}

本文为您介绍如何创建和操作DataFrame对象，以及使用Dataframe完成基本的数据处理。

## DataFrame对象操作 {#section_t1f_pbp_wfb .section}

本文将以[movielens 100K](https://grouplens.org/datasets/movielens/100k/)进行举例。现在已经有了三张表，分别是`pyodps_ml_100k_movies`（电影相关的数据）、`pyodps_ml_100k_users`（用户相关的数据）、`pyodps_ml_100k_ratings`（评分有关的数据）。

1.  当运行环境没有提供ODPS对象时，您需要自己创建该对象。

    ``` {#codeblock_a1s_1p7_czf .language-sql}
    from odps import ODPS
    o = ODPS('**your-access-id**', '**your-secret-access-key**',
             project='**your-project**', endpoint='**your-end-point**')
    ```

2.  创建一个DataFrame对象非常容易，只需传入Table对象即可。

    ``` {#codeblock_d6r_rqu_etc .language-sql}
    from odps.df import DataFrame
    users = DataFrame(o.get_table('pyodps_ml_100k_users'))
    ```

3.  您可以通过`dtypes`属性查看这个DataFrame的字段及字段类型。

    ``` {#codeblock_qrq_9g3_gpy .language-sql}
    users.dtypes
    odps.Schema {
      user_id             int64
      age                 int64
      sex                 string
      occupation          string
      zip_code            string
    }
    ```

5.  如果您不需要看到所有字段，则可以进行如下操作：
    -   从中筛选出一部分字段。

        ``` {#codeblock_bvr_kai_9ub .language-sql}
        users[['user_id', 'age']].head(5)
           user_id  age
        0        1   24

        3        4   24
        ```

    -   只做排除个别字段操作。

        ``` {#codeblock_0se_v8w_hh0 .language-sql}
        users.exclude('zip_code', 'age').head(5)
           user_id  sex  occupation
        0        1    M  technician
        1        2    F       other
        2        3    M      writer
        3        4    M  technician
        ```

    -   排除掉一些字段，通过计算得到一些新的列。例如将`sex`为`M`的置为True，否则为False，并取名为`sex_bool`。

        ``` {#codeblock_lcq_hfn_xc4 .language-sql}
        users.select(users.exclude('zip_code', 'sex'), sex_bool=users.sex == 'M').head(5)
           user_id  age  occupation  sex_bool

        1        2   53       other     False
        2        3   23      writer      True
        3        4   24  technician      True
        4        5   33       other     False
        ```

6.  查看男用户和女用户的个数。

## Dataframe数据处理 {#section_z6t_wdm_hjz .section}

本文使用，请您首先[下载鸢尾花数据集](http://docs-aliyun.cn-hangzhou.oss.aliyun-inc.com/assets/attach/125930/cn_zh/1563431411948/iris.csv)保存到本地。本文使用DataWorks PyODPS节点功能，详情请参见[PyODPS节点](../../../../cn.zh-CN/使用指南/数据开发/节点类型/PyODPS节点.md#)。

1.  创建测试数据表

    使用DataWorks表管理功能新建表，完成后单击**DDL模式**。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344008951888_zh-CN.png)

    输入建表语句如下，完成后提交表。

    ``` {#codeblock_1yh_hkm_vde .lanuage-sql}
    CREATE TABLE `pyodps_iris` (
        `sepallength` double COMMENT '片长度(cm)',
        `sepalwidth` double COMMENT '片宽度(cm)',
        `petallength` double COMMENT '瓣长度(cm)',
        `petalwidth` double COMMENT '瓣宽度(cm)',
        `name` string COMMENT '种类'
    ) ;
    ```

2.  上传测试数据

    单击**导入**。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344008951890_zh-CN.png)

    输入表名称后，上传您刚下载的数据集。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344008951891_zh-CN.png)

    单击**按位置匹配**后导入数据。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344008951896_zh-CN.png)

3.  新建一个PyODPS节点，用于存放和运行代码。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344009051906_zh-CN.png)

4.  输入代码后，单击运行，运行后可在下方**运行日志**处查看结果。

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/21173/156344009051908_zh-CN.png)

    代码详情如下。

    ``` {#codeblock_g7i_mci_gk9 .language-python}
    from odps.df import DataFrame
    from odps.df import output
    iris = DataFrame(o.get_table('pyodps_iris')) #从ODPS表创建DataFrame对象iris。
    print iris.head(10)
    print iris.sepallength.head(5) #打印iris部分内容。
    
    #使用自定义函数求iris的两列之和。
    print iris.apply(lambda row: row.sepallength + row.sepalwidth, axis=1, reduce=True, types='float').rename('sepaladd').head(3)
    
    #指定函数的输出名称和类型。
    @output(['iris_add', 'iris_sub'], ['float', 'float'])
    def handle(row):
    #使用yield关键字可返回多行结果。
        yield row.sepallength - row.sepalwidth,row.sepallength + row.sepalwidth
        yield row.petallength - row.petalwidth,row.petallength + row.petalwidth
    #打印前5行结果，axis=1表示列的轴沿着水平的方向。
    print iris.apply(handle,axis=1).head(5)
    ```
